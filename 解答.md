# 求解過程

## 1. 建立電路微分方程式
根據基爾霍夫電壓定律（KVL）：
$$u_L(t) + u_R(t) + u_c(t) = u_s$$

代入電感元件特徵方程式：
$$u_L(t) = L\frac{di_L(t)}{dt}$$

代入電阻元件特徵方程式：
$$u_R(t) = R i_L(t)$$

得到迴路電流方程式：
$$L\frac{di_L(t)}{dt} + R i_L(t) + u_c(t) = u_s$$

代入電容的電流與電壓關係式：
$$i_L(t) = C\frac{du_c(t)}{dt}$$

對時間求導得到：
$$\frac{di_L(t)}{dt} = C\frac{d^2u_c(t)}{dt^2}$$

整理得到以電容電壓 $u_c(t)$ 為變數的二階常係數線性非齊次微分方程式：
$$LC\frac{d^2u_c(t)}{dt^2} + RC\frac{du_c(t)}{dt} + u_c(t) = u_s$$

代入已知參數：
$$R = 8\ \Omega$$

$$L = 1\text{ H}$$

$$C = 0.25\text{ F}$$

$$u_s = 12\text{ V}$$

代入方程式中：
$$(1 \times 0.25)\frac{d^2u_c(t)}{dt^2} + (8 \times 0.25)\frac{du_c(t)}{dt} + u_c(t) = 12$$

$$0.25\frac{d^2u_c(t)}{dt^2} + 2\frac{du_c(t)}{dt} + u_c(t) = 12$$

同乘以 4 得到標準型微分方程式：
$$\frac{d^2u_c(t)}{dt^2} + 8\frac{du_c(t)}{dt} + 4u_c(t) = 48$$

## 2. 求解特徵根與阻尼判斷
令對應的齊次微分方程式為：
$$\frac{d^2u_c(t)}{dt^2} + 8\frac{du_c(t)}{dt} + 4u_c(t) = 0$$

建立特徵方程式：
$$s^2 + 8s + 4 = 0$$

利用一元二次方程式求根公式：
$$s_{1,2} = \frac{-8 \pm \sqrt{8^2 - 4 \times 1 \times 4}}{2 \times 1}$$

$$s_{1,2} = \frac{-8 \pm \sqrt{64 - 16}}{2}$$

$$s_{1,2} = \frac{-8 \pm \sqrt{48}}{2}$$

$$s_{1,2} = \frac{-8 \pm 4\sqrt{3}}{2}$$

解得兩個相異實特徵根：
$$s_1 = -4 + 2\sqrt{3} \approx -0.5359\text{ s}^{-1}$$

$$s_2 = -4 - 2\sqrt{3} \approx -7.4641\text{ s}^{-1}$$

因為特徵根為兩個不相等的負實根（即 $\alpha^2 > \omega_0^2$）：
此系統工作於過阻尼狀態。

## 3. 確定通解結構與穩態值
過阻尼二階系統的通解由非齊次特解（穩態響應）與齊次通解（暫態響應）組成：
$$u_c(t) = u_c(\infty) + A_1e^{s_1t} + A_2e^{s_2t}$$

當 $t \to \infty$ 時，電感相當於短路，電容相當於開路，電容電壓達到直流穩態值：
$$u_c(\infty) = u_s = 12\text{ V}$$

代入特徵根，寫出通用表達式：
$$u_c(t) = 12 + A_1e^{(-4+2\sqrt{3})t} + A_2e^{(-4-2\sqrt{3})t}$$

## 4. 應用初始條件求解待定係數
根據獨立初始條件：
$$u_c(0) = 0\text{ V}$$

$$i_L(0) = 0\text{ A}$$

根據電容電流關係，導出 $t = 0$ 時的電壓變化率初始條件：
$$\left.\frac{du_c(t)}{dt}\right|_{t=0} = \frac{i_L(0)}{C} = \frac{0}{0.25} = 0\text{ V/s}$$

將通解對時間 $t$ 求導，得到電壓變化率表達式：
$$\frac{du_c(t)}{dt} = s_1A_1e^{s_1t} + s_2A_2e^{s_2t}$$

代入 $t = 0$ 的第一個初始條件（電壓初始值）：
$$u_c(0) = 12 + A_1e^0 + A_2e^0 = 0$$

$$A_1 + A_2 = -12$$

代入 $t = 0$ 的第二個初始條件（電流初始值）：
$$\left.\frac{du_c(t)}{dt}\right|_{t=0} = s_1A_1e^0 + s_2A_2e^0 = 0$$

$$s_1A_1 + s_2A_2 = 0$$

將特徵根 $s_1$ 與 $s_2$ 代入上式：
$$(-4+2\sqrt{3})A_1 + (-4-2\sqrt{3})A_2 = 0$$

$$-4A_1 + 2\sqrt{3}A_1 - 4A_2 - 2\sqrt{3}A_2 = 0$$

$$-4(A_1 + A_2) + 2\sqrt{3}(A_1 - A_2) = 0$$

將 $A_1 + A_2 = -12$ 代入：
$$-4(-12) + 2\sqrt{3}(A_1 - A_2) = 0$$

$$48 + 2\sqrt{3}(A_1 - A_2) = 0$$

$$2\sqrt{3}(A_1 - A_2) = -48$$

$$A_1 - A_2 = \frac{-48}{2\sqrt{3}}$$

$$A_1 - A_2 = -8\sqrt{3}$$

建立聯立方程組：
$$A_1 + A_2 = -12$$

$$A_1 - A_2 = -8\sqrt{3}$$

兩式相加（消去 $A_2$）：
$$2A_1 = -12 - 8\sqrt{3}$$

$$A_1 = -6 - 4\sqrt{3} \approx -12.9282$$

兩式相減（消去 $A_1$）：
$$2A_2 = -12 - (-8\sqrt{3})$$

$$2A_2 = -12 + 8\sqrt{3}$$

$$A_2 = -6 + 4\sqrt{3} \approx 0.9282$$

## 5. 最終暫態響應解析式
將待定係數 $A_1$ 與 $A_2$ 代回通解結構中，得到最終解析式：
$$u_c(t) = 12 - (6 + 4\sqrt{3})e^{(-4+2\sqrt{3})t} - (6 - 4\sqrt{3})e^{(-4-2\sqrt{3})t}\text{ V} \quad (t \ge 0)$$

轉換為小數形式的數值表達式為：
$$u_c(t) \approx 12 - 12.9282e^{-0.5359t} + 0.9282e^{-7.4641t}\text{ V} \quad (t \ge 0)$$
